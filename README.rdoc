= Custom Field Extender

Redmine plugin, that extends existing custom field type list. Now we can add any needed logic in custom field behavior.

Now only autoincrement is realized.

= Installation

1. Download the sources and put them to your vendor/plugin directory
2. Restart your server
No migrations needed. Plugin uses Redmine logic to store custom custom_fields

= Translation

English, german and russian translations are available now.
